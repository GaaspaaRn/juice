# Juice Landing Page

A modern, responsive, and high-performance landing page for the Juice party.

## Features
- **Mobile First Design**: Optimized for all devices.
- **Performance**: Built with pure HTML/CSS/JS for maximum speed.
- **Animations**: Smooth scroll, fade-ins, and interactive elements.
- **Theme**: Custom Neon Green (#aff443) and Dark (#0a0a0b) palette.

## How to Run
Simply open the `index.html` file in your browser.

## Customization
- **Styles**: Edit `style.css` to change colors or fonts.
- **Content**: Edit `index.html` to update text and images.
- **Logic**: Edit `main.js` for interaction adjustments.
